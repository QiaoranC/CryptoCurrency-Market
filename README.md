# 加密货币市场实时查询
- 主要基于Angular4+Bootstrap4
- 实时查询全球多大1000多种加密币的市场信息，包括美元价格，及其他主要货币价格，变化量，供应量，市值信息等。
- API来源[CoinMarketCap](https://coinmarketcap.com/)
- 已使用now部署上线 [地址](https://dist-bhcibdbjdk.now.sh/)
